## 📖 Giới thiệu dự án (About The Project)
Dự án ứng dụng kỹ thuật **Process Mining** kết hợp hệ sinh thái **Big Data (Apache Spark, Kafka)** để phân tích hàng triệu dòng nhật ký sự kiện (Event Logs). Mục tiêu là tự động phát hiện điểm nghẽn, tìm nguyên nhân gốc rễ và dự báo thời gian hoàn thành tác vụ, hỗ trợ tối ưu hóa quy trình theo thời gian thực.

### 🎯 Mục tiêu cốt lõi:
1. **Benchmarking Thuật toán:** Thử nghiệm mô hình học máy (GMM, K-Means, Decision Tree, Random Forest).
2. **Kiến trúc Thời gian thực:** Giả lập dữ liệu bằng Apache Kafka và xử lý luồng bằng PySpark Structured Streaming.
